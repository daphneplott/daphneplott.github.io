# daphneplott.github.io

## Projects

### General Coding

Throughout my college classes, I've had the opportunity to work on some basic projects. In C++, 
I designed different data structure implementations, such as Linked Lists, Binary Search Trees,
and AVL. In python, I've worked through a lexer, parser, and relational database model. 
I've implemented algorithms like RSA, the Strongly-Connected-Components algorithm, Dijkstra's,
Convex Hull, and alignment. 

I also had to opportunity to work a lot with agentic engineering. I've explored topics such as
rag, tool calling, prompt engineering, and multi-agent systems. 

### Larger Projects

I've also gotten to work on larger projects in my classes. I'm currently working on a Chess game 
program. The program is run from the command line, but deals with backend requests such as html, 
database access, and websocket. You can view the repository [here](https://github.com/daphneplott/chess). 
I created a startup website that deals with both frontend and backend implementation. The website 
is no longer active, but you can view the repository [here](https://github.com/daphneplott/startup).

My startup was based on a personal project I had worked on previously. The project is a game that 
runs through the command line in python. In the game, you navigate through different rooms in a school 
and play various minigames. Most of the minigames employ different structure and setups, such as 
custom classes, functions, and various logic. 
That project is found [here](https://github.com/daphneplott/FoxfireGame).
